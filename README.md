# JARVIS – Local GenAI Assistant

JARVIS is a local GenAI assistant project built with a FastAPI backend, React frontend, and Ollama for running local large language models. The goal of the project is to create a personal AI dashboard that can chat with the user, respond with voice, process voice input, and generate useful daily summaries.

## Features

- Local LLM chat using Ollama
- FastAPI backend for handling AI requests
- React frontend with a modern assistant dashboard
- Voice input using browser speech recognition
- Text-to-speech responses
- Morning newspaper feature using RSS feeds
- AI-generated summaries of selected news sources
- Modular structure for future tools and automations

## Tech Stack

- Python
- FastAPI
- React
- JavaScript
- Ollama
- Local LLMs
- RSS feeds
- Speech Recognition
- Text-to-Speech

## Project Goal

The purpose of this project is to explore how local LLMs can be used to build a privacy-friendly personal AI assistant. The project focuses on combining backend APIs, frontend interaction, voice features, and AI summarization into one practical assistant interface.
